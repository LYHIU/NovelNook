# 快速验证指南：个人书架管理

**日期**: 2026-06-11 | **关联**: [plan.md](./plan.md)

按以下步骤验证功能是否完整可用。每条对应 spec 中的验收场景。

## 前置条件

- Node.js 18+（后端）
- 网络连接（在线搜索需要）
- Android 手机浏览器 或 PC Chrome（Web App）
- [可选] 微信开发者工具（小程序）

## 启动

```bash
cd backend
npm install
node server.js
# → 📚 私人书架已启动 → http://localhost:3000
```

## 验证场景

### P1 — 搜索书籍

1. **正常搜索**
   - 打开 `http://localhost:3000`，确认页面加载，左栏书架为空、右栏显示搜索表单
   - 平台选择"晋江文学城"，输入 `造神`，点击"搜索"
   - ✅ 应展示匹配的书目列表，每项含书名、作者、链接

2. **按作者筛选**
   - 输入书名 `造神` + 作者 `星海浮萍`，点击"搜索"
   - ✅ 仅展示该作者的作品（如 Baidu 回退通道触发，来源应显示 `baidu`）

3. **非晋江平台**
   - 平台选择"长佩文学"，输入任意书名，点击"搜索"
   - ✅ 应展示官方搜索入口链接，可复制

4. **空输入保护**
   - 不输入书名，直接点搜索
   - ✅ 提示"请输入小说名"

### P2 — 书架管理

5. **一键加入书架**
   - 从搜索结果中点击"加入书架"
   - ✅ 该书出现在左栏书架列表中，信息完整

6. **手动录入**
   - 清空表单，填写书名、作者 → 点击"加入书架"
   - ✅ 新书加入书架

7. **链接导入**
   - 粘贴 `https://www.jjwxc.net/onebook.php?novelid=1234567` → 点击"解析作品 ID"
   - ✅ 自动提取 novelid=1234567，表单填入

8. **导出/导入**
   - 书架有数据时，点击"导出" → 确认下载 JSON 文件
   - 清空书架后，用刚才的文件导入 → ✅ 数据恢复，与导出时一致

### P3 — 标记与笔记

9. **阅读状态**
   - 点击一本书的编辑，切换状态为"看完"→ 保存
   - ✅ 书架列表中该书显示"看完"

10. **笔记持久化**
    - 编辑某书笔记，输入"这是测试笔记"→ 保存
    - 刷新页面，重新打开该书 → ✅ 笔记完整保留

11. **标签**
    - 给书添加标签 `群像, 强剧情` → 保存
    - ✅ 标签以独立样式展示

12. **筛选**
    - 选择筛选器为"想看"
    - ✅ 仅展示状态为"想看"的书籍

### 离线验证

13. **离线浏览**
    - 断开网络（飞行模式），刷新页面
    - ✅ 书架数据正常展示、可筛选、可编辑笔记
    - 点击"在线检索"→ ✅ 提示"当前无网络"

### 删除验证

14. **删除确认**
    - 点击某本书的"删除"
    - ✅ 弹出确认对话框，确认后该书消失，数据不恢复
